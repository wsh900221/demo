HTML、CSS、JS 注释方式
  • HTML 注释：<!-- HTML 注释 -->。使用 < 和 > 是符合HTML标签语法规则的。
  • CSS 注释：/* CSS 注释代码块 */。“/”也可以用书写成实体：⁄
  • CSS 注释：// CSS 注释单行
  • JS 注释：/* JS 注释代码块 */
  • JS 注释：// JS注释单行
